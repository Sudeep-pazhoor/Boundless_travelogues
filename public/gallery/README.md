# Gallery Images

Add your tour photos here. Supported formats:
- JPG/JPEG
- PNG
- GIF
- WebP

**IMPORTANT:** HEIC files (Apple format) are NOT supported by web browsers.
Please convert HEIC files to JPEG before uploading.

The gallery will automatically display all images in a masonry layout that adapts to each image's dimensions.

Simply drop your images into this folder and they'll appear on the gallery page!
